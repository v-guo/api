A better directory of the student body!

Yalies is an redesigned version of Yale's official Face Book, which provides contact information for Yale students. It displays the same data that is shown on the Face Book and Yale Directory, but with improved design, user interface, and security.

Yalies was originally developed by Erik Boesen. However, it has since became a Yale Computer Society project and is ran collectively by Erik Boeson, Tiger Wang, Lawrence Wang, Yiğit Kılıçoğlu, and Valentina Guo. For questions, comments, bug reports, or suggestions, contact erik.boesen@yale.edu.
