About

Yalies is a website that provides data on the Yale student body. It combines data from Yale Face Book and Yale Directory, with enhanced design, user experience, and security. It uses only data that Yale makes public. To learn how to remove your data from Yale's publicly published data set, see here.

Source Code

You may access the source code of this website on GitHub.

Authors

Yalies was originally developed by Erik Boesen. However, it has since became a Yale Computer Society project and is ran collectively by Erik Boeson, Tiger Wang, Lawrence Wang, Yiğit Kılıçoğlu, and Valentina Guo. For questions, comments, bug reports, or suggestions, contact erik.boesen@yale.edu.
